常见问题解答

开始之前

这里没有您需要的信息？请联系我们的团队。

这款库存软件真的免费吗？

是的。MyCompany 采用 Apache 2.0 许可证开源，没有隐藏费用或许可证绑定，并部署在您自己的基础设施上。

可以管理多少个仓库？

数量不限。每个仓库都可包含任意深度的库位树，是否启用库位级跟踪由您决定。

支持条码和移动设备吗？

支持。操作员可在移动设备上进行拣货或移库时扫描商品、库位和批次条码。

可以跟踪批次和序列号吗？

可以。批次适用于所有库存单据，序列号模式会为每件商品创建一个独立批次。

其他系统可以读取当前库存吗？

可以。JSON HTTP API 提供当前库存数据，并支持按类别和库位筛选。
