Kostenlose ERP-Software für kleine und mittlere Unternehmen

MyCompany ist kostenlose ERP-Software im wörtlichsten Sinne: Das gesamte System erscheint unter der Apache-2.0-Lizenz, der Quellcode ist auf GitHub veröffentlicht, und es gibt weder Lizenzgebühren noch versteckte Kosten oder kostenpflichtige Editionen. Es ist keine Testversion und kein Freemium-Angebot — alle Module werden im selben kostenlosen Paket ausgeliefert.

Was „kostenlos“ hier wirklich bedeutet

Viele als kostenlos beworbene ERP-Produkte begrenzen die Zahl der Benutzer, Datensätze oder Module und verlangen dann Geld für den Rest. MyCompany geht einen anderen Weg. Die Software selbst kostet nichts, egal wie Sie sie einsetzen: Die Apache-2.0-Lizenz erlaubt es Ihnen, sie zu betreiben, zu verändern und kommerziell zu nutzen, ohne irgendjemanden um Erlaubnis zu fragen. Das dauerhafte Versprechen des Projekts ist einfach — MyCompany ist und bleibt für immer eine Open-Source-Lösung. Da Sie das System selbst hosten, sind die einzigen echten Kosten Ihr eigener Server und — falls Sie sich dafür entscheiden — die optionalen kostenpflichtigen Entwicklungs- und Support-Leistungen.

Kostenlos zu installieren, läuft auf bescheidener Hardware

Sie können MyCompany in 10-15 Minuten auf Ihrem eigenen Server installieren: ein Einzeiler-Skript unter Linux, ein Installer unter Windows oder Docker. Eine Maschine mit 4 GB RAM genügt, und der Web-Client ist enthalten — Ihr Team arbeitet im Browser. Nach der Installation melden Sie sich mit dem Standard-Administratorkonto an, und ein späteres Update bedeutet nur, eine einzige JAR-Datei zu ersetzen. Unter der Haube basiert das System auf der lsFusion-Plattform (LGPL v3) — Java und PostgreSQL mit vollständigen ACID-Garantien.

Kostenlos heißt nicht eingeschränkt

Verkaufen Sie am Tresen, nicht nur im Backoffice

Das integrierte Kassenmodul bietet Kassierern einen eigenen Bildschirm mit Barcode-Scanning, geteilten Zahlungen, Retouren, Rabattkarten und Befehlen für Fiskalgeräte — ein Geschäft kann seine Kasse also mit demselben kostenlosen System betreiben, das auch Bestände und Buchhaltung verwaltet.

Fragen beantwortet

Was „kostenlos“ bedeutet

Kontaktieren Sie uns

Ist MyCompany wirklich kostenlos?

Ja. Die Software steht unter der Apache-2.0-Lizenz ohne Lizenzgebühren, und der vollständige Quellcode liegt auf GitHub. Sie laden sie herunter, installieren sie auf Ihrer eigenen Hardware und nutzen sie — es gibt nichts zu kaufen.

Gibt es kostenpflichtige Tarife oder Benutzerlimits?

Nein. Es gibt keine kostenpflichtigen Editionen, keine Benutzerlimits und keine gesperrten Funktionen. Kostenpflichtig sind nur die optionalen Leistungen für individuelle Entwicklung und Support, die vom Produkt selbst völlig unabhängig sind.

Darf ich es kommerziell nutzen?

Ja. Die Apache-2.0-Lizenz erlaubt ausdrücklich kommerzielle Nutzung, Veränderung und Weiterverbreitung — Sie können Ihr Unternehmen darauf betreiben und den Code an Ihre Bedürfnisse anpassen.

Wo ist der Haken?

Ehrlich gesagt: Sie hosten es selbst. MyCompany läuft auf Ihrem eigenen Server, diese Maschine liegt also in Ihrer Verantwortung. Dafür gibt es keine Abos, keine Preise pro Benutzer und keine Anbieterbindung — und die Installation dauert in der Regel 10-15 Minuten.

Wo kann ich es vor der Installation ausprobieren?

Eine Live-Demo ist unter demo.lsfusion.org/mycompany verfügbar — öffnen Sie sie im Browser und klicken Sie sich mit Beispieldaten durch die Module.
