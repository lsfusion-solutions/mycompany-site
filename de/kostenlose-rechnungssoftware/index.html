Eine Datenquelle

Die Rechnungsstellung ist mit dem gesamten Unternehmen verbunden

MyCompany teilt Kunden, Artikel, Aufträge, Lieferungen, Zahlungen und Buchungsdaten zwischen Modulen — ohne Doppelerfassung.

Vor dem Start

Was kleine Unternehmen fragen

Prüfen Sie Quellcode, Dokumentation und öffentliche Demo vor Ihrer Entscheidung.

Ist MyCompany-Rechnungsstellung wirklich kostenlos?

Ja. MyCompany ist Open Source unter Apache 2.0, ohne Gebühren pro Benutzer oder Rechnung.

Kann ich es auf dem eigenen Server betreiben?

Ja. MyCompany läuft unter Linux, Windows oder Docker und Ihre Daten bleiben unter Ihrer Kontrolle.

Unterstützt es MwSt., Rabatte und mehrere Währungen?

Ja. Positionen unterstützen Steuern und Rabatte; die Belegart legt Währung und Preise fest.

Kann es Teilzahlungen und Forderungen verfolgen?

Ja. Zahlungen werden vollständig oder teilweise zugeordnet; der offene Saldo bleibt bis zum Ausgleich sichtbar.

Kann eine Rechnung eine Lieferung erzeugen?

Ja. Bei aktivem Lager erstellt eine bestätigte Rechnung die Lieferung manuell oder automatisch.
