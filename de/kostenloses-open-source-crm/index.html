Fragen beantwortet

Vor dem Start

Noch eine Frage? Sprechen Sie mit uns.

Ist MyCompany CRM wirklich kostenlos?

Ja. Open Source unter Apache 2.0 – ohne Lizenzgebühren, versteckte Tarife oder Nutzerlimits.

Gibt es ein Kanban-Board?

Ja. Spalten entsprechen Phasen; das Verschieben einer Karte ändert den Lead-Status.

Unterstützt es Anrufe und E-Mails?

Ja. Kommunikation bleibt beim Lead; Zadarma ergänzt automatische Erfassung, Click-to-Call und WebRTC.

Wie wird ein Lead zum Auftrag oder zur Rechnung?

Erstellen Sie den Beleg aus dem Lead; Kunde, Adresse und Wert werden automatisch übernommen.

Was brauche ich zum Betrieb?

Ein Server mit 4 GB RAM, Linux, Windows oder Docker genügt für den Start.
