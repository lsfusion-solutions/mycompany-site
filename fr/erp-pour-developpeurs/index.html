ERP pour développeurs · tout comme code

Plateforme ERP open source pour développeurs

MyCompany est un ERP prêt pour la production, basé sur lsFusion pour les équipes qui doivent modifier une vraie logique métier. Définissez modèle de données, calculs, contraintes, actions et formulaires dans un langage déclaratif, versionnez chaque changement avec Git et étendez le produit sans maintenir de fork.

Créez des systèmes métier, pas de la plomberie

Un modèle de la base de données à l'interface

lsFusion conserve les parties importantes de l'ERP dans une base de code cohérente. La plateforme déduit persistance, transactions et comportement UI des définitions déclaratives, tandis que l'application reste du texte ordinaire à réviser, brancher et déployer.

LSF

Un langage déclaratif

Décrivez données stockées et calculées, contraintes, événements, actions et formulaires sans disperser les règles entre ORM, services et frontend.

IDE

De vrais outils de développement

Travaillez dans un environnement basé sur IntelliJ IDEA avec complétion, erreurs, recherche des usages et points d'arrêt sur changement de données.

MOD

Des modules plutôt que des forks

Placez la logique de l'entreprise dans un module .lsf séparé avec REQUIRE MyCompany. La mise à jour remplace le JAR produit et conserve l'extension.

Une pile compréhensible

Du source .lsf à un ERP trois tiers

Le serveur lsFusion s'exécute sur la JVM, stocke les données dans PostgreSQL et fournit un client web. La logique déclarative s'exécute près des données, chaque opération est transactionnelle et le module étend toutes les couches.

Questions de développeurs

Avant d'ouvrir l'IDE

Guide de développement

Quelle est la pile technologique ?

Le serveur tourne sur la JVM, les données sont dans PostgreSQL et les utilisateurs travaillent dans un client web. Logique et formulaires sont écrits en lsFusion.

Comment configurer l'environnement ?

Installez IntelliJ IDEA avec le plugin lsFusion, clonez MyCompany et suivez le guide pour lancer localement l'application et le module.

Comment les modules passent-ils les mises à jour ?

Le module .lsf dépend de MyCompany au lieu de le modifier. La mise à jour remplace le JAR produit et conserve votre module.

Existe-t-il une API ?

Oui. MyCompany prend en charge HTTP, JSON, XML, CSV, XLS et DBF ; les systèmes externes n'accèdent pas directement à la base.

Puis-je créer des solutions commerciales ?

Oui. MyCompany est sous Apache 2.0 et la plateforme sous LGPL v3. Il n'y a ni clé de licence ni DRM.

Puis-je tester le langage sans installation ?

Oui. Utilisez l'environnement lsFusion en ligne, parcourez GitHub ou installez MyCompany sur votre serveur.
