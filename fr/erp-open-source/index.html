Code ouvert, contrôle réel

Le code, les données et l'évolution de l'ERP restent entre vos mains

La plupart des entreprises qui évaluent un ERP pour PME font face au même dilemme : un produit abordable « sur étagère » qui les oblige à travailler à sa façon, ou un développement sur mesure qui coûte une fortune. MyCompany est conçu pour les processus non standard. Vous partez de modules prêts à l'emploi qui couvrent les processus métier essentiels, puis vous adaptez le modèle de données, les formulaires et la logique dans le langage lsFusion jusqu'à ce que le système colle à la réalité de votre entreprise.

↗

Véritablement open source : Apache 2.0

L'ensemble du produit est disponible dans un dépôt public sous licence Apache 2.0 — l'une des licences open source les plus permissives qui soient. Vous pouvez utiliser MyCompany commercialement, le modifier et redistribuer vos changements sans payer personne ni demander d'autorisation. Il n'y a pas d'« édition communautaire » amputée des parties utiles : ce que vous voyez sur GitHub est le système complet.

DB

Un socle technique solide

MyCompany est construit sur lsFusion, une plateforme low-code open source (LGPL v3) qui fonctionne avec Java et PostgreSQL et offre des garanties ACID d'origine. Selon l'éditeur de la plateforme, lsFusion monte en charge jusqu'à des milliers d'utilisateurs simultanés et des téraoctets de données — un système adopté quand on est une petite entreprise n'aura donc pas besoin d'être remplacé en grandissant.

✓

Hébergez-le en quelques minutes, mettez à jour avec un seul fichier

Vous pouvez installer MyCompany sur votre propre serveur en 10-15 minutes : un script d'une ligne sous Linux, un installateur sous Windows, ou Docker. Une machine avec 4 Go de RAM suffit pour commencer, et les utilisateurs travaillent via un client web servi sur le port 8080. Mettre à jour le système consiste à remplacer un seul fichier JAR. La documentation est disponible en anglais, en russe et en polonais, et une démo en ligne vous permet d'explorer le système avant toute installation.

Personnalisation sans fork permanent

Un ERP personnalisable — sans forker le code

La personnalisation est ce qui distingue MyCompany de la plupart des ERP open source. Vous ne patchez pas le code source du produit et ne maintenez pas de fork. À la place, vous ajoutez votre propre module lsFusion — un simple fichier texte .lsf qui commence par REQUIRE MyCompany —, vous le déposez sur le serveur et vous pointez le paramètre logics.topModule vers lui. Votre module étend ou remplace le modèle de données, les formulaires et la logique standard, et le produit lui-même reste intact : il n'y a rien à recompiler. Le flux de développement complet, avec un IDE basé sur IntelliJ IDEA et la plateforme lsFusion, est décrit dans la documentation.

Questions et réponses

Avant de commencer

Contactez-nous

Sous quelle licence MyCompany est-il distribué ?

MyCompany est sous licence Apache 2.0, avec le code source complet public sur GitHub. La plateforme sous-jacente lsFusion est également open source, sous LGPL v3.

Puis-je l'utiliser commercialement gratuitement ?

Oui. La licence Apache 2.0 autorise l'utilisation commerciale, la modification et la redistribution sans aucun coût. Pas de frais de licence ni de coûts cachés.

Comment personnaliser MyCompany ?

Vous écrivez votre propre module lsFusion — un fichier .lsf qui fait REQUIRE MyCompany —, vous le placez sur le serveur et vous définissez logics.topModule avec son nom. Les modifications du modèle de données, des formulaires et de la logique vivent dans votre module : les mises à jour du produit n'écrasent pas votre travail et rien n'a besoin d'être recompilé.

Quels modules sont inclus ?

Données de base, Comptabilité, Stocks, Facturation, Achats, Production, Ventes, Commerce de détail/POS, CRM, Projets, RH, Transport et Administration.

Y a-t-il une dépendance au fournisseur ?

Non. Vous hébergez le système vous-même, le code est ouvert et vos données résident dans une base PostgreSQL standard sur votre propre matériel. Si un jour vous cessez d'utiliser nos services, le logiciel et les données restent entièrement les vôtres.

Comment se compare-t-il aux ERP hébergés uniquement chez l'éditeur ?

Les ERP hébergés par l'éditeur peuvent être pratiques, mais c'est l'éditeur qui contrôle vos données, les prix et le calendrier des mises à jour. Avec MyCompany, vous échangez un petit effort d'installation — 10-15 minutes sur votre propre serveur — contre le contrôle total de l'endroit où le système tourne et de la façon dont il évolue.
