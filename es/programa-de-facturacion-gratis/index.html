Antes de empezar

Lo que preguntan las pequeñas empresas

Revisa el código, la documentación y la demo pública antes de decidir.

¿La facturación de MyCompany es realmente gratis?

Sí. MyCompany es código abierto Apache 2.0, sin tarifas por usuario ni por factura.

¿Puedo instalarlo en mi propio servidor?

Sí. MyCompany funciona en Linux, Windows o Docker y mantiene tus datos bajo control.

¿Admite IVA, descuentos y varias monedas?

Sí. Las líneas admiten impuestos y descuentos y el tipo de documento define moneda y precios.

¿Controla pagos parciales y deuda?

Sí. Los pagos se asocian total o parcialmente y el saldo pendiente queda visible hasta el cobro.

¿Una factura puede crear un envío?

Sí. Con Inventario activo, una factura confirmada crea el envío manual o automáticamente.
