ERP para desarrolladores · todo como código

Plataforma ERP open source para desarrolladores

MyCompany es un ERP listo para producción, creado sobre lsFusion para equipos que necesitan cambiar lógica de negocio real. Define el modelo de datos, cálculos, restricciones, acciones y formularios en un lenguaje declarativo, guarda cada cambio en Git y amplía el producto sin mantener un fork.

Crea sistemas de negocio, no infraestructura

Un modelo desde la base de datos hasta la interfaz

lsFusion mantiene las partes importantes del ERP en una base de código coherente. La plataforma deriva persistencia, transacciones y UI de definiciones declarativas, mientras la aplicación sigue siendo texto normal para revisión, ramas y despliegue.

LSF

Un lenguaje declarativo

Describe datos almacenados y calculados, restricciones, eventos, acciones y formularios sin repartir reglas entre ORM, servicios y frontend.

IDE

Herramientas de desarrollo reales

Trabaja en un entorno basado en IntelliJ IDEA con autocompletado, errores, búsqueda de usos y puntos de ruptura al cambiar datos.

MOD

Módulos en lugar de forks

Guarda la lógica de empresa en un módulo .lsf separado con REQUIRE MyCompany. La actualización cambia el JAR del producto y conserva la extensión.

Un stack comprensible

Del código .lsf a un ERP de tres capas

El servidor lsFusion se ejecuta en la JVM, almacena datos en PostgreSQL y sirve un cliente web. La lógica declarativa trabaja cerca de los datos, cada operación es transaccional y el módulo amplía todas las capas.

Preguntas de desarrolladores

Antes de abrir el IDE

Guía de desarrollo

¿Cuál es el stack tecnológico?

El servidor corre en la JVM, los datos están en PostgreSQL y los usuarios trabajan en un cliente web. La lógica y los formularios se escriben en lsFusion.

¿Cómo configuro el entorno?

Instala IntelliJ IDEA con el plugin lsFusion, clona MyCompany y sigue la guía para ejecutar localmente la aplicación y el módulo.

¿Cómo sobreviven los módulos a las actualizaciones?

El módulo .lsf depende de MyCompany en lugar de parchearlo. La actualización sustituye el JAR del producto y mantiene tu módulo.

¿Hay una API?

Sí. MyCompany admite HTTP, JSON, XML, CSV, XLS y DBF; los sistemas externos no necesitan acceder directamente a la base.

¿Puedo crear soluciones comerciales?

Sí. MyCompany usa Apache 2.0 y la plataforma LGPL v3. No hay claves de licencia ni DRM.

¿Puedo probar el lenguaje sin instalar?

Sí. Usa el entorno online de lsFusion, revisa GitHub o instala MyCompany en tu servidor.
