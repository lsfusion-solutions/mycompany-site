Questions before launch

What project teams usually ask

Inspect the source code, documentation and public demo before deciding.

Is MyCompany project management really free?

Yes. MyCompany is open source under the Apache 2.0 license, with no per-user project management license fee.

Does it support Kanban and Gantt charts?

Yes. Tasks can be monitored in a Kanban board and a Gantt chart, as well as in the standard task list.

Can employees track time against tasks?

Yes. Team members can record time entries by project and task, and managers can review timesheets.

Can we configure our own statuses and workflow?

Yes. Project and task types, priorities, tags, statuses and status-transition rules are configurable.

Can projects share data with ERP modules?

Yes. Projects use the same employees, customers and master data as sales, manufacturing, HR and reporting.
