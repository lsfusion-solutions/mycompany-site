Questions, answered

What “free” means

Want help estimating infrastructure or rollout effort? Talk to our team.

Is MyCompany really free ERP software?

Yes. The complete product is licensed under Apache 2.0, with no software license fees. You can download it, self-host it and use it commercially.

Are there paid editions, user limits or feature gates?

No. There is one complete edition. The license does not limit users, records, companies or modules, and there is no enterprise tier holding features back.

What costs should I still expect?

You provide the server and are responsible for operating it. Implementation, custom development and support may also require time or money, but you can use your own team and paid help is optional.

Can a small business run it on modest hardware?

Yes. A server with 4 GB of RAM is enough to start for a small deployment. Installation typically takes 10–15 minutes on Linux, Windows or Docker.

Can I customize the system?

Yes. The source code is public, and the lsFusion platform lets you keep company-specific extensions in a separate module so product updates do not overwrite them.

Where can I test it before installing?

Use the public MyCompany live demo. It opens in a browser with sample data and requires no purchase or installation.
