Free ERP software for small and medium businesses

MyCompany is free ERP software in the most literal sense: the entire system is released under the Apache 2.0 license, the source code is published on GitHub, and there are no license fees, hidden charges or paid editions. It is not a trial and not a freemium tier — every module ships in the same free package.

If you run a small or medium business, this free ERP system gives you inventory, invoicing, purchasing, sales, manufacturing, point of sale, CRM, projects and HR in one place, installed on your own server and fully under your control.

What "free" actually means here

Many products advertised as free ERP software limit the number of users, records or modules and then charge for the rest. MyCompany takes a different approach. The software itself costs nothing, however you use it: the Apache 2.0 license lets you run it, modify it and use it commercially without asking anyone's permission. The project's standing commitment is simple — MyCompany is and will always be an open-source solution. Because you host it yourself, the only real costs are your own server and, if you choose them, optional paid development and support services.

Every module included, no paid add-ons

The free package covers the full scope of a classic ERP for small business:

  • Master Data and Accounting
  • Inventory and Invoicing
  • Purchase and Sales
  • Manufacturing and Transport
  • POS (point of sale)
  • CRM, Projects and HR

There is no "enterprise edition" holding features back — this is the whole system.

Free to install and run on modest hardware

You can have MyCompany installed on your own server in 10–15 minutes: a one-line script on Linux, an installer on Windows, or Docker. A machine with 4 GB of RAM is enough, and the web client is included, so your team works from a browser. After installation you sign in with the default admin login, and updating later means replacing a single JAR file. Under the hood it is built on the lsFusion platform (LGPL v3) — Java and PostgreSQL with full ACID guarantees.

Sell at the counter, not just from the back office

The built-in POS module gives cashiers a dedicated screen with barcode scanning, split payments, returns, discount cards and commands for fiscal devices — so a shop can run its front desk on the same free system that manages its stock and accounting.

Win customers and deliver the work

CRM tracks leads on a kanban board, attaches calls and emails to each lead, and lets you create orders and invoices straight from a lead. The Projects module adds kanban and Gantt views with time entries and timesheets, while HR handles recruitment, attendance with mobile check-in, and payroll calculated from payroll rules and recorded time.

FAQ

Is MyCompany really free?

Yes. The software is licensed under Apache 2.0 with no license fees, and the full source code is on GitHub. You download it, install it on your own hardware and use it — nothing to buy.

Are there paid tiers or user limits?

No. There are no paid editions, user limits or feature gates. The only paid offerings are optional custom development and support services, which are entirely separate from the software itself.

Can I use it commercially?

Yes. The Apache 2.0 license explicitly permits commercial use, modification and redistribution, so you can run your business on it and adapt the code to your needs.

What's the catch?

Honestly: you host it yourself. MyCompany runs on your own server, so you are responsible for that machine. In return you get no subscriptions, no per-user pricing and no vendor lock-in — and installation typically takes 10–15 minutes.

Where can I try it before installing?

A live demo is available at demo.lsfusion.org/mycompany — open it in your browser and click through the modules with sample data.
