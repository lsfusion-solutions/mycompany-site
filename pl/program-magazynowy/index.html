Odpowiedzi na pytania

Zanim zaczniesz

Brakuje tu potrzebnej informacji? Porozmawiaj z naszym zespołem.

Czy program magazynowy jest naprawdę darmowy?

Tak. MyCompany jest open source na licencji Apache 2.0, bez ukrytych opłat i uzależnienia licencyjnego. System działa na Twojej infrastrukturze.

Ile magazynów mogę prowadzić?

Dowolną liczbę. Każdy magazyn może mieć drzewo lokalizacji o dowolnej głębokości, a śledzenie na poziomie miejsca jest opcjonalne.

Czy obsługuje kody kreskowe i urządzenia mobilne?

Tak. Operatorzy mogą skanować kody artykułów, miejsc i partii podczas kompletacji lub przemieszczeń na urządzeniu mobilnym.

Czy mogę śledzić partie i numery seryjne?

Tak. Partie działają we wszystkich dokumentach magazynowych, a tryb seryjny tworzy jedną partię na sztukę.

Czy inne systemy mogą odczytywać bieżące stany?

Tak. JSON HTTP API udostępnia bieżące stany i pozwala filtrować je według kategorii i lokalizacji.
