Ответы на вопросы

Перед началом

Не нашли нужную деталь? Напишите нашей команде.

Программа складского учёта действительно бесплатная?

Да. MyCompany распространяется с открытым кодом по лицензии Apache 2.0 — без скрытых платежей и лицензионной привязки. Система работает на вашей инфраструктуре.

Сколько складов можно вести?

Любое количество. На каждом складе можно создать дерево мест хранения любой глубины, а учёт по ячейкам остаётся необязательным.

Поддерживаются штрихкоды и мобильные устройства?

Да. Сотрудники могут сканировать штрихкоды товаров, ячеек и партий при отборе или перемещении с мобильного устройства.

Можно учитывать партии и серийные номера?

Да. Партии используются во всех складских документах, а серийный режим создаёт отдельную партию для каждой единицы товара.

Могут ли другие системы получать текущие остатки?

Да. JSON HTTP API отдаёт текущие остатки и поддерживает фильтрацию по категории и месту хранения.
