Бесплатная ERP-система для малого и среднего бизнеса

MyCompany — это бесплатная ERP-система с открытым исходным кодом, созданная для малого и среднего бизнеса. Она объединяет в одном решении продажи, закупки, складской учёт, производство, счета и расчёты, розницу с кассовыми местами, CRM, проекты, кадры и транспорт. Система распространяется по лицензии Apache 2.0: без платы за лицензии, без скрытых платежей и без привязки к поставщику.

$0

Открытый код и доработка под ваши процессы

Полный исходный код MyCompany опубликован на GitHub под лицензией Apache 2.0. Решение построено на платформе lsFusion (LGPL v3): Java и PostgreSQL, транзакционная целостность данных (ACID). Собственную логику можно добавлять модулями на языке lsFusion без перекомпиляции системы: достаточно создать файл .lsf и подключить его директивой REQUIRE MyCompany. Если своих разработчиков нет, доступны платные услуги доработки и поддержки — сама система при этом остаётся бесплатной.

Установка на собственном сервере за 10–15 минут

MyCompany устанавливается на ваш сервер: на Linux (Ubuntu 18+, Debian 9+, RHEL 8+, CentOS 8+, Fedora 35+) — одной командой, на Windows — через установщик, также доступен запуск в Docker. Для работы достаточно сервера с 4 ГБ оперативной памяти. После установки система открывается в браузере на порту 8080, вход выполняется под пользователем admin. Обновление сводится к замене одного JAR-файла. Все данные хранятся в вашей базе PostgreSQL — на вашем оборудовании и под вашим контролем.

Продажи, закупки и расчёты

В модуле продаж ведутся заказы клиентов, прайс-листы и типы цен. Поддерживаются условные скидки: если для клиента действует несколько скидок, система автоматически выбирает наиболее выгодную. Заказы связаны с отгрузками, счетами и производственными заказами, поэтому легко проследить всю цепочку сделки.

Вопросы и ответы

Что означает «бесплатно»

Связаться с нами

Действительно ли ERP-система бесплатна?

Да. MyCompany распространяется по открытой лицензии Apache 2.0: платы за лицензии нет, скрытых платежей тоже. Платными являются только необязательные услуги доработки и поддержки.

Есть ли ограничения по числу пользователей?

Лицензия Apache 2.0 не ограничивает ни число пользователей, ни число рабочих мест. Практический предел определяется только мощностью вашего сервера.

Можно ли использовать систему в коммерческих целях?

Да. Apache 2.0 разрешает коммерческое использование, изменение и распространение продукта в компаниях любого размера.

Где хранятся данные и кто ими владеет?

Система работает на вашем собственном сервере, данные хранятся в вашей базе PostgreSQL. Вы полностью владеете данными и не зависите от поставщика: исходный код открыт, и MyCompany всегда будет open-source.

Как доработать систему под свои процессы?

Собственные модули пишутся на языке lsFusion и подключаются без перекомпиляции: создайте файл .lsf с директивой REQUIRE MyCompany. Начать проще всего с раздела «Разработка» в документации, а если нужна помощь — закажите платную доработку у нашей команды.

Как быстро можно начать работу?

Установка на собственном сервере занимает 10–15 минут: на Linux — одной командой, на Windows — через установщик. А чтобы оценить систему без установки, откройте онлайн-демо.
