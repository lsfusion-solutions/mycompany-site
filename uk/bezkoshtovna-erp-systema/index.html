Безкоштовно означає безкоштовно

Без пробного таймера. Без ліміту користувачів.
Без модулів «лише для Enterprise».

MyCompany — однакова повна ERP для будь-якої команди. Ліцензія Apache 2.0 дозволяє встановлювати, комерційно використовувати й змінювати систему без плати за дозвіл.

Нульова вартість ліцензії

Без підписки, продовження чи рахунків за місця. Одна безкоштовна ліцензія діє і для одного користувача, і для всієї команди.

Не потрібно рахувати користувачів і записи

Додавайте працівників, склади, товари, замовлення та роки історії без переходу на дорожчий тариф і без втрати доступу до старих даних.

Потрібні модулі вже включені

Виробництво, POS, CRM, проєкти та HR — не платні доповнення. Репозиторій і пакет встановлення містять увесь продукт.

Безкоштовне не означає спрощене

Серйозна основа ERP.
Без рахунку за ліцензію.

MyCompany працює на Java і PostgreSQL поверх відкритої платформи lsFusion, підтримує ACID-транзакції та містить вебклієнт. Почніть із помірного сервера й масштабуйте інфраструктуру разом із навантаженням.

  • ✓ Повне комерційне використання за Apache 2.0
  • ✓ Дані PostgreSQL під вашим контролем
  • ✓ Встановлення в Linux, Windows і Docker

Єдина система за задумом

Одна дія рухає вперед увесь бізнес.

Підтвердьте продаж — ERP зарезервує товар, створить потребу у виробництві, запропонує закупівлю та підготує бухгалтерські проведення. Усі підрозділи одночасно бачать ті самі дані.

  • Продажі POS зменшують ті самі залишки, які бачать закупівлі
  • Ліди CRM стають замовленнями без повторного введення даних клієнта
  • Виробництво автоматично оприбутковує готову продукцію та фактичну собівартість

Відповіді на запитання

Що означає «безкоштовно»

Потрібна оцінка інфраструктури або робіт із впровадження? Напишіть нашій команді.

MyCompany справді безкоштовна ERP?

Так. Увесь продукт ліцензовано за Apache 2.0 без плати за програмне забезпечення. Його можна завантажити, розмістити на власному сервері та використовувати комерційно.

Чи є платні редакції, ліміти користувачів або функцій?

Ні. Є одна повна редакція. Ліцензія не обмежує користувачів, записи, компанії чи модулі, а окремого Enterprise-рівня немає.

Які витрати все ж можливі?

Ви надаєте сервер і відповідаєте за його роботу. Впровадження, доопрацювання та підтримка також можуть потребувати часу або коштів, але можна працювати власною командою, а платна допомога необов’язкова.

Чи може малий бізнес працювати на помірному сервері?

Так. Для невеликого впровадження достатньо сервера з 4 ГБ RAM. Встановлення в Linux, Windows або Docker зазвичай займає 10–15 хвилин.

Чи можна доопрацьовувати систему?

Так. Вихідний код відкритий, а платформа lsFusion дозволяє зберігати розширення компанії в окремому модулі, щоб оновлення продукту їх не перезаписували.

Де протестувати систему до встановлення?

Скористайтеся відкритим демо MyCompany. Воно відкривається в браузері з тестовими даними й не потребує купівлі чи встановлення.
