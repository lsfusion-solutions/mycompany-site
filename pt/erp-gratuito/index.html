Software ERP gratuito para pequenas e médias empresas

O MyCompany é um software ERP gratuito no sentido mais literal: todo o sistema é disponibilizado sob a licença Apache 2.0, o código-fonte está publicado no GitHub e não há custos de licença, encargos ocultos nem edições pagas. Não é uma versão de avaliação nem um plano freemium — todos os módulos são distribuídos no mesmo pacote gratuito.

Se gere uma pequena ou média empresa, este sistema ERP gratuito dá-lhe inventário, faturação, compras, vendas, produção, ponto de venda, CRM, projetos e RH num único lugar, instalado no seu próprio servidor e totalmente sob o seu controlo.

O que «gratuito» significa realmente aqui

Muitos produtos anunciados como software ERP gratuito limitam o número de utilizadores, registos ou módulos e depois cobram pelo resto. O MyCompany segue uma abordagem diferente. O software em si não custa nada, seja qual for a utilização: a licença Apache 2.0 permite executá-lo, modificá-lo e usá-lo comercialmente sem pedir autorização a ninguém. O compromisso permanente do projeto é simples — o MyCompany é e será sempre uma solução open source. Como é alojado por si, os únicos custos reais são o seu próprio servidor e, se os escolher, os serviços pagos opcionais de desenvolvimento e suporte.

Todos os módulos incluídos, sem extras pagos

O pacote gratuito cobre todo o âmbito de um ERP clássico para pequenas empresas:

  • Dados mestre e Contabilidade
  • Inventário e Faturação
  • Compras e Vendas
  • Produção e Transporte
  • POS (ponto de venda)
  • CRM, Projetos e RH

Não existe uma «edição enterprise» com funcionalidades reservadas — este é o sistema completo.

Gratuito para instalar e executar em hardware modesto

Pode ter o MyCompany instalado no seu próprio servidor em 10–15 minutos: um script de uma linha no Linux, um instalador no Windows ou Docker. Uma máquina com 4 GB de RAM é suficiente, e o cliente web está incluído, pelo que a sua equipa trabalha a partir do navegador. Após a instalação, inicia sessão com o login de administrador predefinido, e atualizar mais tarde significa substituir um único ficheiro JAR. Por baixo, o sistema assenta na plataforma lsFusion (LGPL v3) — Java e PostgreSQL com garantias ACID completas.

Venda ao balcão, não apenas no back-office

O módulo POS integrado dá aos operadores de caixa um ecrã dedicado com leitura de códigos de barras, pagamentos divididos, devoluções, cartões de desconto e comandos para dispositivos fiscais — para que uma loja possa gerir o balcão no mesmo sistema gratuito que gere o stock e a contabilidade.

Conquiste clientes e entregue o trabalho

O CRM acompanha os leads num quadro kanban, associa chamadas e e-mails a cada lead e permite criar encomendas e faturas diretamente a partir de um lead. O módulo Projetos acrescenta vistas kanban e Gantt com registos de tempo e folhas de horas, enquanto o módulo RH trata do recrutamento, da assiduidade com registo móvel e do processamento salarial calculado a partir das regras salariais e do tempo registado.

FAQ

O MyCompany é mesmo gratuito?

Sim. O software está licenciado sob Apache 2.0, sem custos de licença, e o código-fonte completo está no GitHub. Descarrega-o, instala-o no seu próprio hardware e usa-o — não há nada para comprar.

Existem planos pagos ou limites de utilizadores?

Não. Não há edições pagas, limites de utilizadores nem funcionalidades bloqueadas. As únicas ofertas pagas são os serviços opcionais de desenvolvimento à medida e suporte, totalmente independentes do software em si.

Posso usá-lo comercialmente?

Sim. A licença Apache 2.0 permite explicitamente a utilização comercial, a modificação e a redistribuição, pelo que pode gerir o seu negócio com ele e adaptar o código às suas necessidades.

Qual é o senão?

Honestamente: é você que o aloja. O MyCompany corre no seu próprio servidor, pelo que essa máquina é da sua responsabilidade. Em troca, não há subscrições, não há preços por utilizador e não há dependência de fornecedor — e a instalação demora normalmente 10–15 minutos.

Onde posso experimentá-lo antes de instalar?

Está disponível uma demo ao vivo em demo.lsfusion.org/mycompany — abra-a no navegador e percorra os módulos com dados de exemplo.
