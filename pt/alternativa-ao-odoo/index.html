Não é um clone: são compromissos diferentes

Escolha o modelo certo para a empresa

Ambos tratam processos centrais. A diferença prática está em como paga, aloja, estende e mantém o ERP.

OPEN

Um produto open source

Todos os módulos standard MyCompany usam Apache 2.0. Não há edição paga separada nem subscrição por utilizador.

HOST

Controlo da infraestrutura

MyCompany corre no seu servidor ou cloud. Base de dados, cópias e calendário de atualizações ficam sob controlo.

FLOW

Para processos fora do padrão

Estenda dados, lógica e formulários num módulo lsFusion separado, sem alterar o produto.

Perguntas de comparação

Antes de escolher ou migrar

Ver comparação

MyCompany substitui Odoo diretamente?

Não. É outra plataforma e modelo de dados. A migração exige mapear processos, importar dados e recriar lógica própria.

Todos os módulos MyCompany são gratuitos?

Sim. A aplicação standard usa Apache 2.0, uma edição de código e não cobra por utilizador.

Odoo tem opção gratuita?

Sim. Odoo Community é open source e existe One App Free. Standard e Custom usam subscrição por utilizador; confirme a página oficial.

Posso alojar ambos no meu servidor?

Sim. MyCompany é self-hosted. Odoo suporta on-premise, Odoo Online e Odoo.sh.

Como as extensões afetam atualizações?

Módulos MyCompany ficam separados do JAR. Odoo exige código compatível com a versão alvo. Teste ambos antes de produção.

Quando Odoo é mais forte?

Quando valoriza mais o grande ecossistema, cloud gerida, parceiros, apps móveis ou capacidades locais Enterprise.
