Plataforma ERP open source para programadores

MyCompany é um ERP pronto para produção, criado em lsFusion para equipas que precisam de alterar lógica de negócio real. Defina o modelo de dados, cálculos, restrições, ações e formulários numa linguagem declarativa, guarde tudo em Git e estenda o produto sem manter um fork.

Um modelo da base de dados à interface

lsFusion mantém as partes importantes do ERP numa base de código coerente. A plataforma deriva persistência, transações e comportamento da interface de definições declarativas; a aplicação continua texto normal para revisão, branches e implementação.

LSF

Uma linguagem declarativa

Descreva dados guardados e calculados, restrições, eventos, ações e formulários sem dispersar regras por ORM, serviços e frontend.

IDE

Ferramentas de desenvolvimento reais

Use um ambiente baseado em IntelliJ IDEA com conclusão, erros, procura de utilizações e pontos de paragem quando dados mudam.

MOD

Módulos em vez de forks

Coloque a lógica da empresa num módulo .lsf separado com REQUIRE MyCompany. A atualização troca o JAR do produto e preserva a extensão.

Uma stack compreensível

Do código .lsf a um ERP de três camadas

O servidor lsFusion corre na JVM, guarda dados no PostgreSQL e serve um cliente web. A lógica declarativa executa perto dos dados, cada operação é transacional por defeito e o módulo estende todas as camadas.

Perguntas de programadores

Antes de abrir o IDE

Guia de desenvolvimento

Qual é a stack tecnológica?

O servidor corre na JVM, os dados ficam no PostgreSQL e os utilizadores trabalham num cliente web. A lógica e os formulários são escritos em lsFusion.

Como configuro o ambiente?

Instale IntelliJ IDEA com o plugin lsFusion, clone o repositório MyCompany e siga o guia para executar a aplicação e o módulo localmente.

Como sobrevivem os módulos às atualizações?

O módulo .lsf depende de MyCompany em vez de o alterar. A atualização substitui o JAR do produto e mantém o módulo separado.

Existe uma API?

Sim. MyCompany suporta HTTP, JSON, XML, CSV, XLS e DBF; sistemas externos não precisam de acesso direto à base.

Posso criar soluções comerciais?

Sim. MyCompany usa Apache 2.0 e a plataforma LGPL v3. Não existem chaves de licença nem DRM.

Posso testar a linguagem sem instalar?

Sim. Use o ambiente online lsFusion, consulte os repositórios ou instale MyCompany no seu servidor.
