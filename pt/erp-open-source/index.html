Software ERP open source que pode alojar e personalizar

O MyCompany é um ERP open source para pequenas e médias empresas. O código-fonte completo está publicado no GitHub sob a licença Apache 2.0, pelo que pode executar o sistema no seu próprio servidor, estudar como funciona e alterar o que quiser — sem custos de licença, sem custos ocultos e sem dependência de fornecedor.

A maioria das empresas que avalia um ERP para PMEs enfrenta o mesmo dilema: um produto acessível «de prateleira» que a obriga a trabalhar à maneira dele, ou um desenvolvimento à medida que custa uma fortuna. O MyCompany foi concebido para processos não padronizados. Começa com módulos prontos que cobrem os processos de negócio essenciais e depois adapta o modelo de dados, os formulários e a lógica na linguagem lsFusion até o sistema corresponder à forma como o seu negócio realmente funciona.

O que recebe logo à partida

O MyCompany é fornecido como um ERP completo, não como uma framework vazia. O pacote padrão inclui:

  • Dados mestre e Administração
  • Contabilidade: plano de contas, diários, ativos fixos
  • Inventário e Faturação
  • Compras e Vendas
  • Produção
  • Retalho e POS
  • CRM e Projetos
  • RH e Transporte

Genuinamente open source: Apache 2.0

Todo o produto está disponível num repositório público sob a licença Apache 2.0 — uma das licenças open source mais permissivas que existem. Pode usar o MyCompany comercialmente, modificá-lo e redistribuir as suas alterações sem pagar a ninguém nem pedir autorização. Não existe uma «edição comunitária» com as partes úteis retidas: o que vê no GitHub é o sistema completo.

Um ERP personalizável — sem fazer fork do código

A personalização é onde o MyCompany se distingue da maioria do software ERP open source. Não altera o código-fonte do produto nem mantém um fork. Em vez disso, adiciona o seu próprio módulo lsFusion — um simples ficheiro de texto .lsf que começa com REQUIRE MyCompany —, coloca-o no servidor e aponta a definição logics.topModule para ele. O seu módulo estende ou substitui o modelo de dados, os formulários e a lógica padrão, e o produto em si permanece intocado, pelo que não há nada para recompilar. O fluxo de desenvolvimento completo, com um IDE baseado no IntelliJ IDEA e a plataforma lsFusion, está descrito na documentação.

Uma base técnica sólida

O MyCompany é construído sobre o lsFusion, uma plataforma low-code open source (LGPL v3) que corre em Java e PostgreSQL e oferece garantias ACID de origem. Segundo o fornecedor da plataforma, o lsFusion escala para milhares de utilizadores simultâneos e terabytes de dados — para que um sistema adotado enquanto pequena empresa não precise de ser substituído à medida que cresce.

Aloje-o em minutos, atualize com um único ficheiro

Pode instalar o MyCompany no seu próprio servidor em 10-15 minutos: um script de uma linha no Linux, um instalador no Windows ou Docker. Uma máquina com 4 GB de RAM chega para começar, e os utilizadores trabalham através de um cliente web servido na porta 8080. Atualizar o sistema significa substituir um único ficheiro JAR. A documentação está disponível em inglês, russo e polaco, e uma demo ao vivo permite explorar o sistema antes de instalar o que quer que seja.

FAQ

Sob que licença é distribuído o MyCompany?

O MyCompany está licenciado sob Apache 2.0, com o código-fonte completo público no GitHub. A plataforma subjacente lsFusion também é open source, sob LGPL v3.

Posso usá-lo comercialmente de forma gratuita?

Sim. A licença Apache 2.0 permite a utilização comercial, a modificação e a redistribuição sem qualquer custo. Não há taxas de licença nem encargos ocultos.

Como personalizo o MyCompany?

Escreve o seu próprio módulo lsFusion — um ficheiro .lsf que faz REQUIRE MyCompany —, coloca-o no servidor e define logics.topModule com o nome dele. As alterações ao modelo de dados, aos formulários e à lógica vivem no seu módulo, pelo que as atualizações do produto não sobrepõem o seu trabalho e nada precisa de ser recompilado.

Que módulos estão incluídos?

Dados mestre, Contabilidade, Inventário, Faturação, Compras, Produção, Vendas, Retalho/POS, CRM, Projetos, RH, Transporte e Administração.

Há dependência de fornecedor?

Não. Aloja o sistema você mesmo, o código é aberto e os seus dados residem numa base de dados PostgreSQL padrão no seu próprio hardware. Se um dia deixar de usar os nossos serviços, o software e os dados continuam a ser totalmente seus.

Como se compara com sistemas ERP apenas na cloud do fornecedor?

Os produtos ERP alojados pelo fornecedor podem ser convenientes, mas é o fornecedor que controla os seus dados, os preços e o calendário de atualizações. Com o MyCompany troca um pequeno esforço de configuração — 10-15 minutos no seu próprio servidor — pelo controlo total sobre onde o sistema corre e como evolui.
