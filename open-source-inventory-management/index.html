Questions, answered

Before you start

Need a detail that is not covered here? Talk to our team.

Is the inventory software really free?

Yes. MyCompany is open source under the Apache 2.0 license, with no hidden fees or license lock-in. You host it on your own infrastructure.

How many warehouses can I manage?

Any number. Each warehouse can contain a location tree of arbitrary depth, while bin-level tracking remains optional.

Does it support barcodes and mobile devices?

Yes. Operators can scan item, bin and lot barcodes while picking or transferring goods from a mobile device.

Can I track lots and serial numbers?

Yes. Lots work across all inventory documents, and serial mode creates one lot per unit for item-level traceability.

Can other systems read current stock?

Yes. A JSON HTTP API exposes current stock and supports filtering by category and location.
