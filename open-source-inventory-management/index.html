Open-source inventory and warehouse management

MyCompany is free, open-source inventory management software: it tracks stock across any number of warehouses, records every receipt, shipment and transfer, and shows you exactly what is on hand, reserved and available at any moment. The code is published under the Apache 2.0 license, so there are no hidden fees and no license lock-in.

Unlike a standalone stock control app, MyCompany is a complete small-business ERP built on the lsFusion platform (LGPL v3, Java and PostgreSQL, ACID transactions). Your warehouse management software shares one database with purchasing and sales, and you can install it on your own server in 10-15 minutes.

Warehouses, zones and bins

Locations form a hierarchical tree of arbitrary depth: warehouse, zones inside it, bins inside the zones. You can run any number of warehouses, and the structure stays flexible — bin-level detail is optional, so a small stockroom can be a single location while a larger site is broken down as finely as you need.

Every stock movement is a document

All changes to inventory go through documents, which keeps stock control auditable:

  • Receipts for incoming goods and shipments for outgoing ones
  • Transfers between warehouses or bins
  • Stock adjustments — physical counts that post surpluses and shortages
  • Scrap write-offs for damage, loss, defects, or an expiry write-off reason

Barcode picking on mobile devices

When a shipment flagged for picking reaches the Ready state, a picking task is generated automatically. Each line tracks the required versus picked quantity, the operator, and timestamps. Warehouse staff work from a mobile picking form, scanning bin and item barcodes as they go, and can also move goods between locations with mobile transfers.

Lot and serial-number tracking

Lots give you batch tracking across receipts, shipments, transfers, scrap and adjustments. IDs are generated automatically from a prefix and numerator (for example, LOT-000123), and a serial-number mode creates one lot per unit when you need item-level traceability. Scanning a barcode resolves it to the matching lot — and an unknown barcode can create a new one on the spot. Lot barcode labels print to PDF, DOCX, XLSX, RTF or HTML.

Reports built on ledgers you can trust

All figures come from three ledgers — inventory, reservation and cost — so every balance can be traced back to the movements behind it:

  • Inventory report — Expected, Reserved, Available and On-hand quantities as of any date, with filters and a pivot view
  • Inventory valuation — unit and total cost of stock
  • Product moves — the full movement history of an item
  • Cost report — how costs flow through your inventory

Part of a full ERP, open to other systems

Because inventory lives inside the ERP, purchase orders automatically create receipts and sales invoices automatically create shipments — no re-keying. Several sales items can map to one stock unit through a conversion coefficient, and packaging units can be set per document type. Current stock is also available to external systems over an HTTP API that returns JSON and supports filtering by category and location.

FAQ

Is the inventory software really free?

Yes. MyCompany is open source under the Apache 2.0 license, with no hidden fees and no license lock-in. You host it yourself, and installation on your own server takes 10-15 minutes.

How many warehouses and locations can I manage?

Any number of warehouses. Each one is a hierarchical tree of arbitrary depth — warehouse, zones, bins — and bin-level tracking is optional, so you set only as much structure as your operation needs.

Does it support barcodes?

Yes. Operators scan bin and item barcodes in the mobile picking form, a scanned barcode resolves to its lot (unknown barcodes can create one), and lot barcode labels can be printed to PDF, DOCX, XLSX, RTF or HTML.

Can I track lots and serial numbers?

Yes. Batch tracking works across receipts, shipments, transfers, scrap and adjustments, with auto-generated lot IDs. A serial-number mode assigns one lot per unit for item-level traceability.

Does it work on mobile devices?

Yes. Warehouse staff use a mobile picking form with barcode scanning and can perform transfers between locations from a mobile device.

Can other systems read my stock levels?

Yes. Current stock is exposed over an HTTP API that returns JSON and can be filtered by category and location, so external systems can query availability directly.
